# FinConnect

Spring BootでFinConnectというテーマで作るSNSアプリのバックパート

## 機能
### 作成完了
- ユーザーCRUDおよび認証
- コメント, いいねのCRUD
- フォローとアンフォロー
- 掲示物のCRUD
- 各機での連動
- Reactの画面との連動

### 作成予定
- 金融関連ダッシュボード、経済ニュース一覧
- 金融関連日本にはない新しい本人認証機能
- チャット機能
- イベント募集、管理機能

## 技術スタック
- 言語＆フレームワーク：Java、Spring Boot
- データーベース：PostgraSQL、JPA、Hibernate
- セキュリティー：Spring security, JWT, Bcrypt
- その他：POSTMAN
- ネットワークやインフラは現在未定
